# API для проекта Yatube

REST API для социальной сети Yatube, позволяющее работать с публикациями, комментариями и сообществами, а также оформлять подписки на других авторов.

## Технологии:
- Python 3.9+
- Django 3.2+
- Django REST Framework
- Djoser (JWT Аутентификация)

## Как запустить проект:
1. Клонировать репозиторий.
2. Установить зависимости: `pip install -r requirements.txt`
3. Выполнить миграции: `python manage.py migrate`
4. Запустить сервер: `python manage.py runserver`

Документация доступна по адресу `http://127.0.0.1:8000/redoc/`.
